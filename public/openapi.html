Avrillo Kayne West APIs (1.0.0)

Download OpenAPI specification:Download

These API provide the user with the ability to received 5 random quotes from Kayne West. They use the publicly available API at https://api.kayne.rest

Get 5 random quotes

Authorizations:
api_token

Responses

https://localhost/api/quotes

Response samples

  • 200
  • 401
Content type
application/json
[
  • "For me giving up is way harder than trying.",
  • "I wish I had a friend like me",
  • "I'm the best",
  • "Style is genderless",
  • "All the musicians will be free"
]

Refreshes the quotes and returns 5 random Kayne West quotes.

Authorizations:
api_token

Responses

https://localhost/api/refresh
